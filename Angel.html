Wat is de werkervaring?

Doordat ik weer in de eerste ben geplaatst heb ik denk ik meer ervaring met adobe progamma's, maar vind ik photoshop nog steeds best lastig. Van de 3 progamma's vind ik illustrator het makkelijkst. We moesten voor dit project een website maken, een one pager portfolio. Samen met de opleiding software Developers hebben we samen een website gebouwd en hebben we de taken verdeeld. De mediavormgevers maaakte dus de edesign en template en de software Developers voerde het dus uit maar dan om een echte website temaken, Dus met de ervaring daarvan heb ik een template gemaakt voor de one pager.

Voor mij gevoel werkte we best goed samen, ookal was de communitcatie niet het beste aan het begin begonnen we later beter te worden met het bespreken.
